# 医学影像知识库（个人版）

一个可离线使用的个人医学影像学习收藏应用（PWA）。

## 功能
- 10 大系统分类收藏（神经、胸部、腹部、肌骨、心血管、头颈、乳腺、妇产、儿科、介入）
- 全文搜索、标签、收藏置顶
- 支持图片 / PDF / 网页链接 / Markdown 笔记
- PDF 内嵌预览（无需外网）
- 数据导出 / 导入 JSON，可备份迁移
- 支持「添加到主屏幕」，像 App 一样使用

## 使用
手机浏览器打开后，点「添加到主屏幕」即可像原生 App 使用。
所有数据保存在本地浏览器，不上传服务器。
